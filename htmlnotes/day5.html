Forms

HTML Forms are used to collect user input on a webpage.

syntax

Types of Form Inputs in HTML

  • Text (type="text") → Used for single-line text input.
  • Password (type="password") → Hides characters for secret input like passwords.
  • Email (type="email") → Accepts email addresses with validation.
  • Number (type="number") → Allows numeric input with optional min/max values.
  • Date (type="date") → Provides a date picker for selecting a date.
  • Time (type="time") → Lets users select a time (hours & minutes).
  • Checkbox (type="checkbox") → Allows multiple selections from options.
  • Radio (type="radio") → Allows only one selection from a group.
  • File (type="file") → Used to upload files from the device.
  • Range (type="range") → Provides a slider control to select a numeric value.
  • Color (type="color") → Opens a color picker for selecting a color.
  • Hidden (type="hidden") → Stores data in the form without showing it.
  • Submit (type="submit") → Sends the form data to the server.
  • Reset (type="reset") → Resets all form fields to default values.
